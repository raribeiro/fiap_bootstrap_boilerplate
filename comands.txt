Link do meu perfil no github: https://github.com/raribeiro
Link para o repositório: https://github.com/raribeiro/fiap_bootstrap_boilerplate

## comandos para iniciar o projeto localmente e subir para o github

$ curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.37.2/install.sh | bash

$ source ~/.bashrc

$ nvm install node

$ npm install -g yo
$ npm install -g generator-gulp-webapp

$ mkdir ativ_cap4_07_02_a_11_03

$ npm install -g gulp

$ yo webappgulp-webapp

$ git init

$ git remote add origin git@github.com:raribeiro/fiap_bootstrap_boilerplate.git

$ git add -A

$ git branch -m master

$ git commit -m

$ git push origin master

## gerando arquivos finais para deploy

$ gulp build
